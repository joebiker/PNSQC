# Quality Jam 2026 - Integrate MCP Into Your Workflow

Welcome! 

Let’s get started integrating the Model Context Protocol (MCP) into your workflow.

We will work with a variety of MCP servers to connect your quality apps and find ways to do your job better and faster. Come with your apps and we will connect in to your scenarios too.

This session will use the following tools. Please arrive with these, or equivalent applications, installed and ready to use:

 - Chrome web browser
 - Microsoft Visual Studio Code
  - GitHub Copilot Chat 
  - microsoft/playwright-mcp
 - GitHub Copilot CLI
 - Google Gemini CLI

We may also utilize online tools. If you do not already have accounts, you may want to consider signing up for the following:

 - Postman.com
 - Trello.com
 - Github.com
