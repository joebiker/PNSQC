## System Path
**Set the path for the current user, ie: Joe. (super important)**
Open Environment Variables
User variables (top section)
Find Path, select it, then click Edit...
Copy the following path:
C:\Users\Joe\AppData\Roaming\npm\
On the Edit environment variable screen click New
Paste the path, click OK.
